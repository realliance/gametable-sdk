# Gametable SDK

A C++ SDK for using [libmahjong](https://github.com/realliance/libmahjong) Player Controllers directly in Gametable Network Matches.

## Dependencies

- CMake
- pistache (AUR: `pistache-git`)
- `rapidjson`
- `spdlog`

## Getting Started

```
# Bootstrap build/
cmake -S . -B build

# Build library
cmake --build build
```
